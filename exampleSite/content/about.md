I'm Thomas Laurenson — a developer who thinks the web works best when it is fast,
readable, and honest about what it is.

letsblaze is the theme I built for my own site. It has no JavaScript, no external
stylesheets, no web fonts, and no CDN dependencies. Every page is plain HTML with
a small inline style block. That is the entire stack.

The name comes from the idea that a site should be genuinely fast — not fast after
optimisation, but fast by default because there is nothing to slow it down.

If that sounds like what you want, letsblaze might be the right theme for you.
